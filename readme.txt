== Description ==

Makes it easy to display a link to the Creative Commons license you desire in posts.

It adds the following features:

* Options to add a link to your Creative Commons license under all posts

Read more about this plugin on the [Creative Commons Suite plugin](http://coenjacobs.net/wordpress/plugins/creative-commons-suite) page.

== Installation ==

1. Upload the `creative-commons-suite` directory to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Set the settings for displaying the link underneath posts in Options panel provided

== Frequently Asked Questions ==

= Where can I find a overview of the Creative Commons licenses? =

Choose your [Creative Commons license](http://creativecommons.org/choose/) on the official website.
